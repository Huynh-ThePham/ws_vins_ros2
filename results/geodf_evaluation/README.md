# GeoDF-Weighted evaluation artifacts

Paper evaluation outputs for branch `paper/geodf-weighted-vins-2026`.
All tables and figures here are **generated by scripts** from real benchmark runs —
do not hand-edit numbers.

## Reproducible pipeline (N-repeat, single build)

```bash
source scripts/setup_ws.bash
export VIODE_ROOT=/path/to/viode
export EUROC_ROOT=/path/to/EuRoC

# Optional: rebuild weighted binaries once before any benchmark
# REBUILD=1 bash scripts/run_viode_n5_prepare.sh 5

# --- Prepare (no VIO trials) ---
bash scripts/run_viode_n5_prepare.sh 5    # 12 VIODE bags + GT + manifest
bash scripts/run_euroc_n3_prepare.sh 3    # 5 EuRoC bags + manifest

# --- Run (baseline + weighted, N trials per cell) ---
FORCE=1 bash scripts/run_viode_n5.sh 5
FORCE=1 bash scripts/run_euroc_n3.sh 3

# --- Optional: VIODE detection (weight < threshold vs GT masks) ---
bash scripts/run_viode_detection_prepare.sh

# --- Postprocess: tables + figures ---
bash scripts/postprocess_paper_artifacts.sh
```

## Artifacts produced (under this directory)

| File | Content |
|------|---------|
| `PAPER_RESULTS_N5.md` + `.json` | VIODE ATE/RPE mean±std, baseline vs weighted |
| `EUROC_REPEAT_N3.md` + `.json` | EuRoC static-safety mean±std |
| `STATS_TESTS.md` + `.json` | Welch/MWU significance, effect sizes, CI |
| `RUNTIME_TABLE.md` + `.json` | Per-frame geo cost from weighted trials |
| `RESOURCE_TABLE.md` | Peak RSS / CPU (if `resource.json` present in trials) |
| `DETECTION_EVAL_VIODE.md` | Mask-level detection lift (optional) |
| `verify_report.json` | Trial integrity gate |
| `figures/viode_ate_delta_n5_gray.*` | ATE improvement bar chart |
| `figures/viode_detection_lift_gray.*` | Detection lift (if detection runs exist) |
| `figures/viode_trajectories_gray.*` | Qualitative trajectory overlays |

Raw per-trial data: `results/viode_repeat/`, `results/euroc_repeat/`.
Detection feature dumps: `results/viode_detection/*_weighted/`.
